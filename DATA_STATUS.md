# حالة البيانات في قاعدة البيانات

## ✅ البيانات الوهمية (Fake Data)

تم إنشاء Seeder (`FakerDataSeeder`) لملء قاعدة البيانات بالبيانات التالية:

### البيانات المتوفرة:

1. **مستخدم إداري واحد:**
   - Email: `admin@inventory.com`
   - Password: `password`
   - Type: `admin`

2. **10 سائقين:**
   - Emails: `driver1@inventory.com` إلى `driver10@inventory.com`
   - Password: `password`
   - Type: `driver`
   - أسماء عربية

3. **8 فئات:**
   - أكياس التسوق
   - أكياس القمامة
   - أكياس الطعام
   - أكياس السحاب
   - أكياس التجميد
   - أكياس النفايات الطبية
   - أكياس التغليف
   - أكياس الزراعة

4. **40+ منتج:**
   - منتجات بلاستيكية (أكياس)
   - صور من Unsplash
   - أسعار وأوصاف عربية

5. **مخزون المستودع:**
   - كل منتج له كمية في المستودع

6. **مخزون السائقين:**
   - توزيع عشوائي للمنتجات على السائقين

7. **30 عملية بيع:**
   - فواتير كاملة مع items
   - تواريخ عشوائية (آخر 30 يوم)
   - عملاء بأسماء عربية

## كيفية ملء البيانات:

```bash
# حذف جميع البيانات وإعادة إنشائها
php artisan migrate:fresh --seed

# أو فقط ملء البيانات (إذا كانت الجداول موجودة)
php artisan db:seed --class=FakerDataSeeder
```

## ملاحظات:
